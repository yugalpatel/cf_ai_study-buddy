# cf_ai_study-buddy

An AI-powered flashcard study app built on Cloudflare.

## What it does
Paste your notes → AI generates flashcards → Quiz yourself via chat → Progress is saved.

## Tech stack
- **LLM**: Llama 3.3 via Cloudflare Workers AI
- **Compute**: Cloudflare Workers
- **Memory**: Durable Objects (session) + Workers KV (flashcard storage)
- **Frontend**: Cloudflare Pages
